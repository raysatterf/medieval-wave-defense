# Medieval Wave Defense

A simple web-based tower defense game with a medieval theme. Build and upgrade towers to defeat waves of enemies. Works in modern browsers on desktop and mobile.

## Playing

Open `index.html` in your browser. Click on the map to place towers (cost 50 gold). Click an existing tower to upgrade it. Defeat the incoming waves before you run out of life.

## Development

The game uses plain HTML5, CSS and JavaScript. Logging output appears in the on-screen console and browser console. Audio feedback is generated via the Web Audio API.
